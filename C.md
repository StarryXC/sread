## 变量存储类型

```
变量存储类型
    auto
    register
    static
    extern
auto 自动变量
	所有局部变量定义的默认类型
	只能用在函数内
	只能修饰局部变量
register 寄存器变量
	存储在寄存器中的变量
	变量的最大尺寸等于寄存器的大小
	不能使用一元的 '&' 运算符操作
	一般只用于需要快速访问的变量
	该变量不一定存于寄存器中，与硬件实现有关
static 静态类型
	修饰局部变量：在程序的生命周期内保持局部变量的存在
	修饰全局变量：使变量的作用域限制在声明它的文件内
	全局变量默认 static
extern 外部类型
	修饰的全局变量对所有程序文件可见
```
