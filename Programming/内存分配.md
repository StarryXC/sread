### C

```
calloc() 分配一个二维储存空间
free() 释放已分配空间
malloc() 分配空间
realloc() 改变已分配空间的大小
```

### Android

```
MemoryFile 共享内存
try {
    MemoryFile memoryFile = new MemoryFile("mf-name", 500);
    byte[] buffer = new byte[1024];
    memoryFile.writeBytes(buffer, 0, 0, buffer.length);
    memoryFile.close();
} catch (IOException e) {
    e.printStackTrace();
}
```
