> Thinking

```
<html> # 文档根标签
<head> # 头部
<title> # 文档标题
<body> # 
<base> # 默认超链接设置
<meta> # 元数据
description 文档描述
keywords 页面关键词
author 作者

content-type 文档类型
expires 过期时间
refresh 刷新
set-cookie Cookie 设置
```
